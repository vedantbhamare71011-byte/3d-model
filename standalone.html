Cannot reach local Python Backend

HTML files running inside a web browser cannot directly execute PyTorch/CUDA tasks due to browser security sandboxes. To actually use your NVIDIA GPU, you must run the Python backend on your Windows machine:
  1. Open terminal/command prompt in the project folder
  2. Install python dependencies: pip install fastapi uvicorn python-multipart opencv-python numpy
  3. Run the backend: python -m uvicorn backend.main:app --host 127.0.0.1 --port 8000
  4. Click "Generate 3D Model" again.
